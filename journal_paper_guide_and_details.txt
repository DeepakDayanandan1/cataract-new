NOTE ON EQUATIONS: ARE THEY REALLY NECESSARY?
--------------------------------------------------------------------------------
Yes, for reputable journals (like IEEE, Wiley, Springer, Elsevier), relying solely 
on "I imported a Deep Learning model and it got 95% accuracy" is often rejected 
for being a "black-box" approach. 

Adding the mathematical equations serves three essential purposes in a journal:
1. Academic Rigor: It proves you understand the underlying mechanics of the CNNs, not just the code.
2. Reproducibility: It clearly defines the exact loss function (Weighted Cross-Entropy) and optimization techniques used, allowing other researchers to replicate your work.
3. Page Length & Formatting: It fills out the Methodology section structurally, which is expected in high-impact medical imaging papers.

You don't need to invent new math, but presenting standard equations (like Convolution, ReLU, Softmax, and your specific Weighted Loss) proves technical depth.

5. PROPOSED METHODOLOGY [THIS IS THE MOST IMPORTANT SECTION]

A. Dataset Description
- Fundus Dataset: Multiclass grading (Normal, Mild, Moderate, Severe).
- Slit-Lamp Dataset: Maturity grading (Normal, Immature, Mature).

B. Image Preprocessing and Augmentation
Images resized to 224x224 pixels. A robust augmentation pipeline was applied (Scaling Factor = 4x):
- Geometric Transformations: Random rotations, horizontal, and vertical flipping.
- Intensity Transformations: Color jittering to account for different camera lighting conditions.

C. Handling Class Imbalance
Clinical datasets inherently suffer from severe class imbalance. To mitigate this, Class Weights were calculated and applied to the Weighted Cross-Entropy Loss function. 
(Example Fundus Weights: Normal 1.0, Mild 3.5, Moderate 1.8, Severe 1.0).

D. Deep Learning Architectures
1. Main Proposed Model: DenseNet-169 (For Both Fundus and Slit-Lamp):
DenseNet-169 connects each layer to every other layer in a feed-forward fashion. By utilizing 169 layers, it significantly deepens the feature extraction capabilities compared to DenseNet-121, strengthening feature propagation and encouraging extensive feature reuse for complex biological structures (like retinal vessels and lens opacities). We replaced the final classifier with a custom dense block:
(Linear = 512 units) -> (ReLU activation) -> (Dropout = 0.5) -> (Linear output = 4 or 3 units).

2. Comparative Baseline Models:
To validate the efficacy of the proposed model, we conducted a comparative analysis against established architectures:
- ResNet-50: Provided a baseline for residual learning with 50 layers.
- DenseNet-121: Evaluated to demonstrate the necessity of the increased depth (169 layers vs 121 layers) in handling fine-grained ocular details.

PART 3: ALL THE MATHEMATICAL EQUATIONS YOU NEED
--------------------------------------------------------------------------------
*Copy and paste these equations over to your Word/LaTeX document. Journals love mathematical rigor.*

1. Convolution Operation
F(i, j) = (I * K)(i, j) = Σ_m Σ_n I(i + m, j + n) K(m, n)

2. Rectified Linear Unit (ReLU)
f(x) = max(0, x)

3. Softmax Function (For Classification)
P(y_i | z) = exp(z_i) / Σ_{j=1}^{K} exp(z_j)

4. Weighted Categorical Cross-Entropy Loss [Crucial for your paper]
L = - Σ_{i=1}^{N} w_{y_i} log( P(y_i | x_i) )
Where:
- N is the batch size.
- w_{y_i} is the class weight assigned to the true class of instance x_i.
- P(y_i | x_i) is the predicted probability for the actual true class.

5. Cosine Annealing Learning Rate
η_t = η_min + 0.5 * (η_max - η_min) * (1 + cos( T_cur / T_max * π ))

6. Evaluation Metrics
Accuracy = (TP + TN) / (TP + TN + FP + FN)
Precision = TP / (TP + FP)
Recall (Sensitivity) = TP / (TP + FN)
F1-Score = 2 * (Precision * Recall) / (Precision + Recall)
